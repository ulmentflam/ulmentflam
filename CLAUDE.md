# CLAUDE.md

This file provides guidance to Claude Code when working with code in this repository.

## Project Overview

GitHub profile README for [ulmentflam](https://github.com/ulmentflam) (Evan Owen). This is a special repository — its `README.md` renders on the GitHub profile page.

## Structure

- **`README.md`** — The profile README rendered at github.com/ulmentflam
- **`assets/banner.svg`** — Custom SVG banner (1280x320, dark navy + teal circuit-trace design)
- **`assets/stats.svg`** — GitHub stats card (auto-generated by GitHub Actions)
- **`assets/top-langs.svg`** — Top languages card (auto-generated by GitHub Actions)
- **`.github/workflows/stats.yml`** — Daily workflow to regenerate stats SVGs

## Brand

- **Primary background**: `#1a1f2e` (dark navy)
- **Accent color**: `#2dd4bf` (teal)
- **Text**: white (`#ffffff`)
- **Badge style**: `flat-square` for tech stack, `for-the-badge` for social links

## External Services

- **shields.io** — Tech stack and social badges
- **readme-typing-svg** (`readme-typing-svg.demolab.com`) — Animated typing effect
- **streak-stats** (`streak-stats.demolab.com`) — Contribution streak card
- **komarev** — Profile view counter

## Maintenance

- Blog posts are manually listed in the README (between `BLOG-POST-LIST` comments)
- Stats cards are regenerated daily at 3 AM UTC via GitHub Actions
- To update featured projects or bio, edit `README.md` directly
- The banner SVG is self-contained with no external dependencies
